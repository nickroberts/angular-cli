# ng generate enum

## Overview
`ng generate enum [name]` generates an enumeration

## Options
<details>
  <summary>app</summary>
  <p>
    `--app` (alias: `-a`) _default value: 1st app_
  </p>
  <p>
    Specifies app name to use.
  </p>
</details>
